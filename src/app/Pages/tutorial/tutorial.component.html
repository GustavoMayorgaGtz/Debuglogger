Vista de los Sensores
En esta sección puedes observar los sensores donde se muestra los datos históricos de temperatura así como el dato actual, dato mínimo y dato máximo;
Opciones
Las opciones que se encuentran en la parte derecha de la página web, te ayudaran a organizar mejor la información que requieres así como visualizarlas de una manera diferente.
Seleccion de sensor
Una vez hagas clic en la lista desplegable, puedes ver los 8 sensores disponibles, selecciona el sensor del cual quieras saber la información.
Modifica el dato de advertencia
Este valor es una medida de protección para los sensores, si el valor actual supera este valor se dispara una advertencia que se encargara de enviar un correo electrónico notificando lo que sucede al encargado. El valor se puede modificar haciendo un clic en el icono de editar que se encuentra en la parte derecha del bloque.
Ocultar datos
En caso de que solo se quieran observar ciertos valores tienes la posibilidad de seleccionar los datos que quieres ver en la sección de gráficas.
Descargar datos
Tienes la opción de descargar los datos en formato Excel, se mostrarán todos los datos recopilados del sensor que seleccionaste.
